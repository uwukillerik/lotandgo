# Юридические документы Lot&Go (дипломный проект)

Документы опубликованы в веб-интерфейсе и доступны по прямым ссылкам.

## Список документов

| Документ | URL | Назначение |
|----------|-----|------------|
| Политика конфиденциальности | `/legal/privacy` | Обработка и защита ПДн (152-ФЗ) |
| Пользовательское соглашение | `/legal/terms` | Правила использования платформы |
| Согласие на обработку ПДн | `/legal/personal-data` | Текст согласия субъекта данных |
| Политика cookie | `/legal/cookies` | Использование файлов cookie |
| Публичная оферта | `/legal/offer` | Условия сделок на аукционах |

Исходные тексты: `lib/legal-content.ts`.

## Реализация согласия пользователя

1. **Регистрация** — обязательные чекбоксы:
   - принятие пользовательского соглашения и публичной оферты;
   - согласие на обработку персональных данных и ознакомление с политикой конфиденциальности.

2. **API** — поля `acceptTerms` и `acceptPrivacy` валидируются схемой `registerSchema` (`shared/schemas.ts`).

3. **База данных** — при регистрации сохраняются метки времени:
   - `users.terms_accepted_at`
   - `users.privacy_accepted_at`

   Миграция: `drizzle/0005_legal_consent.sql`.

4. **Cookie-баннер** — компонент `components/cookie-consent.tsx`, ссылка на `/legal/cookies`.

5. **Футер** — ссылки на все документы (`components/site-footer.tsx`).

## Для пояснительной записки

В разделе «Защита персональных данных» можно указать:

- соответствие архитектуры требованиям 152-ФЗ (согласие, политика, ограничение доступа, хэширование паролей);
- разграничение ролей user/admin;
- учебный характер системы (демо-кошелёк, отсутствие реальных платежей в тестовом режиме);
- дисклеймер на каждой юридической странице о демонстрационном назначении.

## Запуск миграции

```bash
pnpm db:migrate
```

После миграции новые регистрации сохраняют дату принятия документов.
